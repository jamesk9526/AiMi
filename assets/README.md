# Icon Placeholder

This folder should contain the application icon files:

- `icon.png` - 512x512px PNG for Windows and Linux
- `icon.icns` - ICNS file for macOS
- `icon.ico` - ICO file for Windows

You can create these using online tools or image editors.

Recommended sizes:
- PNG: 512x512px, 256x256px, 128x128px, 64x64px, 32x32px, 16x16px
- ICO: Multi-size ICO file with all the above sizes
- ICNS: Standard macOS icon format

For now, the app will use the default Electron icon until you add your custom icons here.
