# AiMi Random Images

This folder contains images that AiMi can randomly send to users during conversations.

## Adding Images

To add more images:
1. Place image files (jpg, png, gif) in this folder
2. The app will automatically detect and use them
3. Recommended image size: 800x600 or similar aspect ratio
4. Keep images appropriate and aligned with AiMi's personality
